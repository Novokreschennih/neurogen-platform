## 2. Шаг 1: Создание бота в @BotFather

Открой Telegram и найди официального бота для создания ботов — **@BotFather** (он с синей галочкой).

**Пошаговая инструкция:**

1. Нажми `ЗАПУСТИТЬ` или отправь команду `/newbot`
2. Придумай имя (Name). Оно будет видно в заголовке чата. Скопируй вариант ниже:

```
NeuroGen | ИИ-Ассистент 🤖
```

3. Придумай юзернейм (Username). Это будет ссылка на твоего бота. Он должен быть уникальным и заканчиваться на `bot`. Например:

```
sethubble_biz_bot
```

4. BotFather выдаст длинное сообщение с **API TOKEN**. Он выглядит примерно так:

```
1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
```

> ⚠️ **БЕЗОПАСНОСТЬ ТОКЕНА:** Это ключ от твоего бота. Отправь его только нашему обучающему боту. Никогда не передавай его третьим лицам!

## 3. Шаг 2: Настройка внешнего вида

Чтобы бот выглядел профессионально, прямо в @BotFather отправь несколько команд:

**🎨 Команда `/setuserpic` (Аватарка)**

Отправь команду, выбери своего бота и загрузи картинку. Используй стильные арты из наших материалов или найди в интернете по запросу «cyberpunk AI logo».

**📝 Команда `/setdescription` (Описание до старта)**

Отправь команду, выбери бота и вставь этот текст:

```
Привет! Я твой нейросетевой бизнес-ассистент. Я покажу, как автоматизировать привлечение клиентов и получать пассивный доход, оцифровав сарафанное радио. Жми СТАРТ 👇
```

**ℹ️ Команда `/setabout` (Коротко о себе)**

Отправь команду, выбери бота и вставь:

```
Создаю автономные IT-бизнесы и ИИ-воронки. Жми /start для запуска! 🚀
```
