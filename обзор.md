# 📄 Документация проекта: NeuroGen Bot & Ecosystem

## 1. Общее описание
**NeuroGen** — это гибридная SaaS-платформа и маркетинговая надстройка над блокчейн-платформой **SetHubble**. 
- **Миссия:** Помочь пользователям (Агентам) и бизнесу (Онлайн/Офлайн) строить партнерские сети через ИИ-автоматизацию.
- **USP (Уникальное торговое предложение):** «Ваш цифровой клон». Система создает для партнера персонального ИИ-бота и набор инструментов (Промо-кит, лендинги), которые продают систему 24/7.

## 2. Технологический стек
- **Backend:** Node.js (Yandex Cloud Functions).
- **Database:** YDB (Yandex Database) — NoSQL, Serverless-режим.
- **AI Core:** Интеграция с Polza.ai / OpenRouter (основная модель: **DeepSeek v4 Flash**).
- **Frontend:** Статический сайт (Eleventy), Tailwind CSS, Vanilla JS.
- **Интеграции:** SetHubble API (реферальная система), HubblePay (платежи в USDT).

## 3. Архитектурные особенности

### 3.1. Omni-Channel Resolver (`omni_resolver.js`)
Уникальный механизм «склейки» профилей. Если пользователь:
1. Зашел в Web-чат через браузер.
2. Оставил Email.
3. Позже перешел в Telegram или VK.
Система объединяет все его идентификаторы (`web_id`, `tg_id`, `vk_id`, `email`) в **единую запись в БД**. Это позволяет сохранять прогресс обучения и реферальную привязку между всеми мессенджерами.

### 3.2. AI Engine v3.x (`ai_engine.js`)
«Мозг» системы. 
- **Контекстная осведомленность:** ИИ знает, на каком шаге воронки находится пользователь, его баланс NeuroCoins и статус (FREE/PRO).
- **Knowledge Map:** ИИ имеет доступ к базе знаний только по тем модулям, которые пользователь уже прошел (защита от спойлеров).
- **Reasoning:** Оптимизирован под модели рассуждения (DeepSeek), имеет расширенные лимиты `max_tokens` (1500-2000) и таймауты (15-30 сек).

### 3.3. Multi-Bot Core
Система поддерживает два режима работы в мессенджерах:
1. **Центральный бот:** Быстрый старт для новичков (FREE). Работает через реферальные параметры в ссылке.
2. **Личные боты-клоны:** Доступны только для **PRO**-партнеров. Партнер подключает свой токен из @BotFather, и система разворачивает на нем его персональную воронку с его аватаркой и именем.

## 4. Воронка продаж и Геймификация

### 4.1. Путь пользователя
1. **Join Page:** Лендинг с захватом Email.
2. **Обучение (3 модуля):** Стратегия, Онлайн (настройка бота), Офлайн (B2B).
3. **NeuroCoins:** За прохождение уроков и ввод секретных слов начисляется внутренняя валюта.
4. **Tripwire (PRO-статус):** Основной продукт. Цена: **$20** (если есть 100 монет) или **$40** (базовая). Дает 50% комиссии и доступ к ИИ-лаборатории.
5. **Scaling (Rocket/Shuttle):** Дорогие тарифы для масштабирования сети (глубина до 10 уровней и Бинар).

### 4.2. ИИ-Подписка (SaaS)
Дополнительный продукт — доступ к «мозгу» для клиентов партнера.
- **Trial:** 3 дня бесплатно для новых партнеров.
- **Цена:** $3/мес за использование системного ключа ИИ.
- **Лимиты:** FREE — 3 сообщения/день, PRO — 30 сообщений/день. Безлимит — при вводе своего API-ключа в Промо-ките.

## 5. Ключевые компоненты Frontend

### 5.1. Promo-Kit (`/promo-kit/`)
Личный кабинет партнера.
- **Дашборд:** Визуализация воронки (Клики -> Лиды -> Доход).
- **Mode Toggle:** Переключатель режимов (Агент / Онлайн / Офлайн). Меняет реф-ссылки и контент на лету.
- **O2O Tools:** Генератор рекламных PDF-флайеров с QR-кодами для печати.
- **AI Constructor:** Тонкая настройка характера бота (Промпт-инжиниринг) и выбор LLM-моделей.
- **AI Copwriter:** Генератор уникальных постов для соцсетей.

### 5.2. Web Chat (`/ai/`)
Полноценный интерфейс мессенджера в браузере. Поддерживает всю логику кнопок сценария и свободное общение с ИИ.

## 6. Безопасность
- **JWT (JSON Web Token):** Используется для авторизации во всех веб-приложениях системы (Промо-кит, CRM, PDF-генератор). Срок жизни — 24 часа.
- **Payload Validation:** Строгая проверка всех входящих данных через `validator.js`.
- **Deduplication:** Защита от повторной обработки одних и тех же вебхуков Telegram/VK.

## 7. Технические эндпоинты (API)
- `action=web-chat`: Обработка действий в веб-чате.
- `action=generate-post`: Изолированный ИИ для написания постов.
- `action=get_partner_link`: Получение данных для Промо-кита.
- `action=update_ai_settings`: Сохранение настроек ИИ.
- `action=get_public_config`: Публичные данные для лендинга.
