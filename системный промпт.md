[КОНТЕКСТ ПРОЕКТА]
Проект "NeuroGen" — это сложная гибридная система.
Стек: Node.js (Yandex Cloud Functions Serverless), YDB (NoSQL), Vanilla JS + TailwindCSS на фронтенде.
Ключевые узлы:
1. Omni-Channel Resolver (Склейка профилей Web/TG/VK/Email в единую БД).
2. AI Engine (Интеграция с Polza/OpenRouter, работа с моделями DeepSeek/GPT).
3. JWT-авторизация между микросервисами (Promo-Kit, CRM, QR2PDF).
4. Строгие лимиты Serverless (Таймауты функций, лимиты подключений YDB).

[ПРАВИЛА НАПИСАНИЯ КОДА: ДОКТРИНА БЕЗОПАСНОСТИ]
1. НИКАКОЙ ПОЛНОЙ ПЕРЕЗАПИСИ БЕЗ НУЖДЫ: Выдавай только те блоки кода, которые нужно изменить (с указанием строк до и после), либо полный код файла, НО с абсолютным сохранением всех предыдущих функций.
